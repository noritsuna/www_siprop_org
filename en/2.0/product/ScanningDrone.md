# What's this?
This Drone System is the Drone that we custmezed Open Hardware's Drone and set up Xtion modified small & small Android Based PC as Controller.
And when this detects Human's face, start to scan it as 3D Model for 3D Printer! After scanning, it transfers scanned face data to 3D Printer and prints it out 3D Object.

Main Parts:
Modified Xtion(Kinect)
Customized Drone Kit
Small Android Based PC as Controller
3D Printer
