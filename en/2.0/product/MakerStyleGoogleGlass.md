## Setup Linaro/Android & Custom Kernel
1. Setup [Linaro/Android13.03](https://releases.linaro.org/13.03/android/panda/) from binary
    1. Install Linaro Tools
<br>

```
 $ sudo add-apt-repository ppa:linaro-maintainers/tools
 $ sudo apt-get update
 $ sudo apt-get install linaro-image-tools
```
    1. Get Binaries
        1. [boot.tar.bz2](https://releases.linaro.org/13.03/android/panda/boot.tar.bz2)
        1. [system.tar.bz2](https://releases.linaro.org/13.03/android/panda/system.tar.bz2)
        1. [userdata.tar.bz2](https://releases.linaro.org/13.03/android/panda/userdata.tar.bz2)
    1. Install Linaro/Android to SD Card.
        1. Change /dev/sdX to your SD Card.
<br>

```
sudo ./linaro-image-tools/linaro-android-media-create --mmc /dev/sdX --dev panda --boot boot.tar.bz2 --system system.tar.bz2 --userdata userdata.tar.bz2
```
    1. Install graphics libraries
<br>

```
 $ wget http://people.linaro.org/~vishalbhoj/install-binaries-4.0.4.sh
 $ chmod a+x install-binaries-4.0.4.sh
 $ ./install-binaries-4.0.4.sh
```
1. Setup USB FS
    1. Edit "/init.rc"
 Add a following line under "on boot"
```
mount usbfs none /proc/bus/usb -o devmode=0666
```
