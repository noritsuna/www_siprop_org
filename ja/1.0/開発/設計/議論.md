# 議論の進め方
こちらの[開発/設計](../設計.md)の各部分について、個別に議論していきます。<br>
まずは、大枠を検討し、方向性が決まったら、その方向性についてさらに細分化し、より詳細の検討をしていく感じとなります。<br>
だいたい、下記のような感じになることを想定しています。

```
機能検討→機能詳細/機能実装方法
```

※議論が終了次第、[開発/設計](../設計.md)にマージしていきます。<br>
