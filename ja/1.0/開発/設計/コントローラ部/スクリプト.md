# 目的
- [FlatSIPメッセージ](../オブジェクト/FlatSIPメッセージ.md)と[コントロールメッセージ](../オブジェクト/コントロールメッセージ.md)から、ロード対象の[シーケンス制御/吸収部](../制御モジュール/シーケンス制御/吸収部.md)を特定する
    - ルータやリゾルバに近い動作
- [シーケンス制御/吸収部](../制御モジュール/シーケンス制御/吸収部.md)やコントロール部の動作を決定する
    - 全体のコントロールを制御する
- また、[コントロールメッセージ](../オブジェクト/コントロールメッセージ.md)に必要な情報を付加する

# 実装方針
- まずは、クラスとして実装
- のちに、独自言語などにして、定義し直す

# 定義方法
- 保留で、c=0を使いたいかa=sendonlyを使いたいかなどは、このスクリプトで定義できるようにする。
