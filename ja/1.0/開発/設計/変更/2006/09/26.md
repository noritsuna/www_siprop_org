## Transportからメッセージの扱い
- PacketAcceptorクラスとTransportとStackの間に挟み、byte[]をMessageContextクラスに変換する。
>  class PacketAcceptor {
>    void run() {
>      byte[] rawPacket = Transport.recv();
>      MessageContext msg = new MessageContext(rawPacket);
>      StackRouter.doDispatch(msg);
>    }
>  }
