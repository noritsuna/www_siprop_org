# クラス図
クラスの概要図を示す
- B2BUAクラス
    - 全体の管理を行う
    - UAに対して、管理権限を持つ
- Routerクラス
    - スクリプト部に相当するクラス
    - パケットやメッセージのルーティングに対して、権限を持つ
- UAクラス
    - 制御モジュールに相当するクラス
    - シーケンス(UA)動作に対して、権限を持つ
- Stackクラス
    - StackやTransportに対して、権限を持つ

- Resolverクラス
    - 送信先を管理するクラス
    - inner,outer条件も管理する
- Filterクラス
    - IP制限やAuthチェックなどをするクラス
    - エラー応答なども生成する
- MessageContextクラス
    - 1UAインスタンスで持ち回されるメッセージを保持しているクラス
