# 各種ソフトウェア入手先
- J2SE1.4 http://java.sun.com/downloads/
- J2ME Wireless Toolkit 2.2 http://java.sun.com/products/sjwtoolkit/ja_download-2_2.html
    - JavaDOC http://java.sun.com/j2me/docs/pdf/cldc11api.pdf
    - W-ZERO3用デバイスファイル http://book.mycom.co.jp/support/e2/willcomd/
        - s23.zip というファイルになります。
        - 「WILLCOM 携帯アプリプログラミングブック」という書籍のサンプルとなりますので、本書籍を購入の上、ダウンロードしてください。((本書籍と本プロジェクトに、関連はありません。))
- Eclipse http://www.eclipse.org/
    - Eclipse J2ME Plugin http://eclipseme.org/

# セットアップ
- J2SE1.4セットアップ
    - インストーラに従いインストールします。
- J2ME Wireless Toolkit 2.2セットアップ
    - インストーラに従いインストールします。
    - patch は、インストール先ディレクトリ解凍するだけOKです。
- Eclipse
    - インストーラに従いインストールします。
- Eclipse J2ME Plugin
    1. Eclipse上で、[Help]→[ソフトウェア更新]→[検索とインストール]を選択します。
    1. [インストールする新規フィーチャーを検索]→[新規リモートサイト]で、「http://eclipseme.org/updates/」を入力してください。
    1. あとは、インストーラに従いインストールを続行します。
    1. インストール終了後、[ウィンドウ]→[設定]を開きます。
    1. [Antenna JAR]と[WTK Root]に、「J2ME Wireless Toolkit」のインストールパスを入れて適用します。

# 環境構築
1. Eclipse の svn Plugin にて、[ダウンロード](../ダウンロード.md)を参考に、ソースを取得します。
1. [実行]→[構成および実行]→[Wireless Tookkit Emulator] にて、新規構成します。
1. [Executable]の[Midlet]に、[org.siprop.xxx]を指定し、適用します。

- 以上にて、Eclipse上にて、ソース～実行までが可能となります。
