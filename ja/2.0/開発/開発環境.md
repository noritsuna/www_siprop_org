# 各種ソフトウェア入手先
- J2SE5.0以上 http://java.sun.com/downloads/
- Eclipse http://www.eclipse.org/
- Maven2 or 各種ライブラリ
    - MINA1.1.0以上 http://mina.apache.org/
        - slf4j1.3.0以上 http://www.slf4j.org/
    - JainSIP-Parser1.2以上 http://www.siprop.org/m2-repo/jain-sip/JainSIP-Parser/1.2/JainSIP-Parser-1.2.jar
        - backport-util-concurrent3.0以上 http://dcl.mathcs.emory.edu/util/backport-util-concurrent/ 

# セットアップ
- J2SE5.0セットアップ
    - インストーラに従いインストールします。
- Eclipse
    - インストーラに従いインストールします。

# 環境構築
1. Eclipse の svn Plugin にて、[ダウンロード](../ダウンロード.md)を参考に、ソースを取得します。
1. Eclipse の Maven2 Plugin にて、ライブラリを取得します。

- 以上にて、Eclipse上にて、ソース～実行までが可能となります。
