# 雷電3号
## 新設計
### MessageRouter部
- APIとメッセージのみ独自仕様
    - 内部的には、下記のものを利用する
        - [ServiceMix](http://servicemix.apache.org/home.html)
        - [Srping Integration](http://www.springframework.org/spring-integration)
- 理由は、Androidなどへのポーティングを考えて、省電力のための機構を組み込むため

### Agent部
- OSGiベースの実装を取り入れる
    - こちらにも、省電力のための機構を組み込む。実質AndroidのActivityのAPIに近いものとなる。
